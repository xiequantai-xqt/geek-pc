## 技术栈

- react@17.0.2
- react-router@5.3.0
- antd@4.18.6
- ajax 请求库：axios
- 富文本编辑器：react-quill
- CSS 预编译器：sass
- 项目搭建：React 官方脚手架 create-react-app

## 简介

实现了文章后台管理的功能，axios 拦截，配合路由钩子函数实现权限控制、登录、登出功能

接口文档：http://geek.itheima.net/api-pc.html

## Build Setup

### 安装依赖

```shell
npm install
```

### 运行项目（将自动在浏览器打开 http://localhost:3000 ）

```shell
npm start
```
